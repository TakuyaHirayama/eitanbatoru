# ⚔️ Word Dungeon: Legend Pro

英単語学習を「冒険」に変える、没入型RPGスタイルのクイズアプリケーション。

## 🌐 試用URL
https://eitanbatoru-qyhwhko7xdisblj4ibjwbj.streamlit.app/

## 🌟 主な機能
- **10体討伐ゴール制**: 10体目に到達すると「LAST BOSS」が出現。高HP・高火力の強敵を撃破してクリアを目指します。
- **究極の判定精度**: SupabaseのUUID（ID）ベースで照合を行うため、文字列の空白や表記揺れによる誤判定が一切ありません。
- **コンボ & ダメージシステム**: 連続正解でコンボが発生し、攻撃力が可視化されます。ミスをすると答えが大きく表示され、即座に学習できます。
- **豪華リザルト画面**: クリアランク(S/A/B)、討伐数、最大コンボ、クリアタイムを詳細にレポート。
- **リアルタイム単語管理**: 管理画面から即座に新しい英単語をデータベースへ追加・更新可能。

## 🛠️ 技術スタック
- **Frontend**: Streamlit
- **Backend/DB**: Supabase (PostgreSQL)
- **Design**: Cyberpunk / Dark Mode CSS

## 📝 使い方
1. 「🔥 潜入開始」ボタンでダンジョンへ。
2. 表示される単語の意味を4択から選び、敵のHPを削ります。
3. 自分のHPが0になる前に10体のモンスターを倒せばミッション完了。
4. 終了後に表示される「復習リスト」で苦手な単語を確認できます。
